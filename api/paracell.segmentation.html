get_seg_path(fov: FOV, update_fov: bool = False) → str[source]¶

Gets path segemetations are saved to for an FOV.

The path is the first path that exists (in the following order): 1. FOV’s seg_path 2. Segmentor’s seg dir 3. Plate’s output_dir (if FOV has a parent plate) 4. FOV’s parent dir

The dirs have the FOV prefix and _seg.npy appended.

Parameters:

  • fov (FOV) – FOV to get the segmentation path for.

  • update_fov (bool, optional) – Whether or not to update the FOV’s seg_path with the calculated path. Defaults to False.

Returns:

Path to segmentation file.

Return type:

str

Raises:

ValueError – If all places where the seg path could be set are not set.

class paracell.segmentation.transforms.RandomRotate(p=0.5, max_angle=180.0)[source]¶

Bases: Module

Randomly rotate the (image, mask, flows) by an angle in [0, max_angle). Applies: 1) A spatial rotation to the image, mask, and flows. 2) A vector rotation to the flows’ (flowY, flowX).

The mask is rotated with nearest-neighbor interpolation. The image & flows use bilinear interpolation.

forward(sample)[source]¶
sample is expected to be (image, annotations):

  • image: a torch.Tensor of shape [C, H, W]

  • annotations[“flows”]: a np.ndarray or torch.Tensor, shape [3, H, W]

    flows[0] = flowY, flows[1] = flowX, flows[2] = cellprob (or something similar)
