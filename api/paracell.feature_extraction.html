class paracell.feature_extraction.vision.ColocalizationAnalysis[source]¶

Bases: CellAnalysisMixin

Analysis module for stats about the colocalization of channels.

analyze_cell(cell: Cell, fov: FOV) → None[source]¶

Adds colocalization-related features to the cell’s stats.

Precondition:

  • There are at least two channels in the FOV.

  • The cell has been segmented.

Params:

cell (Cell): Cell to add stats to. fov (FOV): FOV containing the cell.

Additional Info:

  • Pearson and Mander’s overlap coefficients default to 0 for empty and identical images.

  • Channels are added in alphabetical order.

class paracell.feature_extraction.vision.CompartmentAnalysis[source]¶

Bases: CellAnalysisMixin

analyze_cell(cell: Cell, fov: FOV)[source]¶

Adds the statistics about the intensities of the area localized in the nucleus, cytoplasm and remaining periphery, for each channel.

Precondition:

  • The cell has been segmented.

  • A nucleus is required in the nuclear channel. Otherwise no stats will be added.

Parameters:

  • cell (Cell) – Cell to add the statistics to.

  • fov (FOV) – FOV the cell belongs to.

static calculate_intensity_features(cell: Cell, intensity_image: ndarray, feature_name: str)[source]¶

Generate basic intensity features given a certain input intensity image. For example, you may want to input an image of intensities that are solely in the nucleus.

Features will NOT be scaled.

Parameters:

  • cell (Cell) – The cell whose profile will be modified.

  • intensity_image (np.ndarray) – Image containing intensities (0-65535).

  • feature_name (str) – Name of the feature.
